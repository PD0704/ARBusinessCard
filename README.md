\# AR Business Card



An AR-powered business card app built with Unity, Vuforia Cloud Recognition, and Firebase.



Scan a physical card to see the owner's full profile in augmented reality.

\## Tech Stack

\- Unity 2022.3 LTS

\- AR Foundation + Vuforia Cloud Recognition

\- Firebase (Firestore + Storage + Cloud Functions)

\- ZXing.Net for QR scanning
